# Print Images

Logging is way too boring and harder to read on the console. Especially when there is a dumpster of messages.
Thankfully, now we can print images on the console instead!

```java
const const var img MyImage = ImagineTheImage.fromFs(""""./path/to/rick_roll.png"""")
print(MyImage)!
```

It will load the image to the memory (`img` keyword ensures it lives there forever),
And then every time an `img` is passed as a parameter to `print` function, it will generate an ASCII art from it (without memoizing, because we value your memory stack ♥).
